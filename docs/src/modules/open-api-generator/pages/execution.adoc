Execute *open-api-generator* and enter the requested inputs, starting by the *generation mode*.

[source,bash,subs="+attributes"]
----
mvn exec:java@open-api-generator
----

TIP: To override the *operationId sanitize mode* at generation time, pass the property on the command line (see xref:open-api-generator:index.adoc#operation-id-sanitize-mode[Open API Generator - Configuration - Operation ID Sanitize Mode] for available modes):

[source,bash,subs="+attributes"]
----
mvn exec:java@open-api-generator -Dopen-api-operation-id-sanitize.mode=letters-only
----

TIP: Select *generation mode* using the up/down arrows `&#x2191;` `&#x2193;` and afterwards `intro` to proceed `&#x21b2;` to next step.
